# Week 2 – Finance Notes (28 June 2026)

## Sharpe Ratio
- Formula: (Return - Risk-Free Rate) / Volatility
- Higher = better risk-adjusted return
- Used to compare investments across different asset classes

## Alpha
- Excess return over benchmark
- Positive alpha = manager adds value
- Negative alpha = manager destroys value

## Risk-Adjusted Returns
- Sharpe: excess return per unit of total risk
- Sortino: excess return per unit of downside risk
- Calmar: excess return per unit of max drawdown
