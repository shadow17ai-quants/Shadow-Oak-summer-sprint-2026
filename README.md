## Key Features

- **Data pipeline**: 1000+ instruments, gap/outlier detection, corporate action handling
- **Alpha signals**: Cross-sectional momentum, Bollinger mean reversion, carry
- **Backtester**: Event-driven, full Indian cost stack, 3 slippage models
- **Risk metrics**: Sharpe, Sortino, Calmar, Omega, drawdown, VaR
- **Statistical tests**: Jarque-Bera, Shapiro-Wilk, ADF, Ljung-Box, bootstrap CI
- **Live dashboard**: Streamlit + PostgreSQL trade journal
