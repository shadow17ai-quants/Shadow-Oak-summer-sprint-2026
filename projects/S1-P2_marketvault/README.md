
📊 MarketVault — Live Market Dashboard & Strategy Builder
Version

Project
MarketVault (formerly NifftyVault) is an advanced, real-time financial dashboard built with Streamlit. It aggregates live market data via Yahoo Finance to provide powerful tracking, portfolio benchmarking, and a custom rule-based backtesting engine across Indian and global markets.

✨ Core Features
📈 Analytics & Visualization
 * Live Overview: Interactive price charts overlaid with an RSI(14) indicator and a 20-day annualized volatility metric.
 * Top Movers: Real-time leaderboard ranking the top 10 instruments by return over the last 10 bars.
 * Monthly Heatmap: A clean visual grid mapping historical monthly returns to spot seasonal trends.
💼 Portfolio Tracking & Risk Management
 * Portfolio Mode: Benchmark multiple instruments simultaneously using a normalized growth-of-$1 comparison.
 * Risk Metrics: Instant calculation of per-instrument Sharpe Ratio, volatility, and maximum drawdown.
 * Smart Alerts: Sidebar thresholds that trigger real-time visual alerts for RSI overbought/oversold levels and severe drawdowns.
🧪 Backtesting & Strategy Engine
 * Strategy Builder: Construct custom rules by combining Moving Average Crossovers, RSI, and Bollinger Bands using customizable AND / OR logic. Benchmark your equity curve directly against Buy-and-Hold.
 * Strategy Comparison: Configure and pit two independent trading strategies side-by-side to evaluate outperformance.
 * Data Export: One-click CSV and report generation for raw price data, portfolio summaries, and backtest logs.
🎨 Design System & Theme
The interface utilizes a tailored, professional aesthetic designed for low eye strain and rapid signal recognition:
 * 🟢 Deep Pistachio: Utilized for structural headers and section navigation.
 * 🌿 Mid Pistachio: Used for primary chart lines and positive trend indicators.
 * 🔴 Maroon: Reserved strictly for borders, alert badges, and sell-side signals.
🚀 Getting Started
Prerequisites
Ensure you have Python installed alongside the required dependencies (streamlit, yfinance, pandas, plotly).
Installation & Execution
Run the dashboard via PowerShell or your standard terminal interface:
# Clone the repository and navigate to the root directory
cd marketvault

# Launch the Streamlit application
streamlit run src/marketvault/dashboard.py
