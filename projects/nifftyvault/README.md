# 🔺 NifftyVault – Nifty 50 Data Analyzer

**Shadow Oak Capitals – S1-P2**

A quantitative data pipeline that downloads, analyses, and visualises Nifty 50 market data. Computes log returns, volatility, and prepares data for Sharpe ratio and alpha analysis.

---

## 📌 Features

- 📥 Download 5 years of Nifty 50 data via yfinance
- 📊 Compute daily log returns and simple returns
- 📈 Summary statistics (mean, std, min, max)
- 🗂️ Export data to CSV
- 📉 Visualise price history

---

## 🗂️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python 3.11+ | Core language |
| yfinance | Market data download |
| Pandas | Data manipulation |
| NumPy | Numerical operations |
| Matplotlib | Visualisations |
